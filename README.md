### Instalacion

#### Requisitos (Node y npm)

Versiones recomendadas:
- Node.js 20 LTS
- npm 10 (viene incluido con Node 20)

Opciones de instalacion:

**macOS / Linux (nvm)**
1. Instala nvm: https://github.com/nvm-sh/nvm
2. Instala Node 20 y activalo:
   - `nvm install 20`
   - `nvm use 20`
3. Verifica versiones:
   - `node -v`
   - `npm -v`

**Windows (nvm-windows)**
1. Instala nvm-windows: https://github.com/coreybutler/nvm-windows
2. Instala Node 20 y activalo:
   - `nvm install 20`
   - `nvm use 20`
3. Verifica versiones:
   - `node -v`
   - `npm -v`

1. Clona el repositorio:
   - `git clone https://github.com/cmontedonico/V0-Gonzalo-Celorio.git`
2. Entra a la carpeta del proyecto:
   - `cd V0-Gonzalo-Celorio`
3. Instala dependencias:
   - `npm install`

### Ejecucion

- Desarrollo:
  - `npm run dev`
- Build de produccion:
  - `npm run build`
- Servidor de produccion:
  - `npm run start`
